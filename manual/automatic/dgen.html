Synopsis

Document parser and generator

Description

Scan source code files, build document model and facilitate generation of documentation files (HTML, tex etc.)

At present exclusively dedicated to scanning

  • dLabPro scripts (with and w/o DOM),
  • C/C++ source files,
  • Pearl scripts.

May be extended in the future.

Example


  DGen iUD
  "test.itp" "uasr" NULL iUD -scan
  <%
    <html>
    <head>
      <title>${iUD.spl[0,1]}/${iUD.spl[1,1]}</title> # UASR package/scriptname
    </head>
    <body>
      <table>
  %>
  iUD.txt "<tr><td>" "</td><td>" "</td></tr>" iUD -table
  <%
      </table>
    </body>
  %>
  "test.html" iUD -write

Synopsis

Adds table data to HTML page

Description

Each row of the table is preceeded by a prefix string sPre. Between the data cells the delimiter string sDel will be placed. The suffix string sSuf is appended after the last cell of each row.

Example

To generate a simple HTML table from a data instance idTab you would write:


    UasrDoc iUD
    ...
    iUD <% <table> %>
    idTab "<tr><td>" "</td><td>" "</td></tr>" iUD -table
    iUD <% </table> %>

Options

   /bare   Do not place the prefix before/the suffix after the first/last row.
